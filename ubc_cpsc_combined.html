大一入门课，建立编程基础。CPSC 110是CS专业主入口，CPSC 121建立离散数学基础。

CPSC 100
Computational Thinking
计算思维
3 学分其他
Meaning and impact of computational thinking, problem solving, testing, debugging, and how computers work. No prior computing experience needed.
计算思维的意义与影响、问题求解、测试、调试及计算机工作原理。无需编程经验。
前置: 无前置要求
CPSC 103
Introduction to Systematic Program Design
系统程序设计导论
3 学分核心编程
Computation for systematic problem solving in non-CS disciplines; introductory programming skills. Not for credit with CPSC 110 or APSC 160.
面向非CS学科的系统化问题求解计算；入门编程技能。与CPSC 110/APSC 160互斥学分。
前置: 无前置要求
CPSC 107
Systematic Program Design
系统程序设计
3 学分核心编程
Fundamental computation and program structures. Continuing systematic program design from CPSC 103.
基础计算与程序结构。延续CPSC 103的系统化程序设计。
前置: CPSC 103
CPSC 110
Computation, Programs, and Programming
计算、程序与编程
4 学分核心编程
Fundamental program and computation structures. Introductory programming skills. Computation for info processing, simulation, modelling, and interacting with the world.
基础程序与计算结构。入门编程技能。用于信息处理、模拟、建模及与世界交互的计算。这是CS专业的主要入门课。
前置: 无前置要求
CPSC 121
Models of Computation
计算模型
4 学分理论/算法
Physical and mathematical structures of computation; Boolean algebra, combinational logic, proof techniques, functions, sequential circuits, sets/relations, finite state machines.
计算的物理与数学结构：布尔代数、组合逻辑、证明技巧、函数、时序电路、集合/关系、有限状态机。
前置: Principles of Math 12 或 Pre-calculus 12
并修/注意: 并修: CPSC 107 或 CPSC 110

CPSC 203
Programming, Problem Solving, and Algorithms
编程、问题求解与算法
3 学分核心编程
Analysis of complex algorithmic problems using modern programming; topics like Voronoi Diagrams, Markov Chains, Bin Packing, Graph Search.
使用现代编程分析复杂算法问题：Voronoi图、马尔可夫链、装箱问题、图搜索。
前置: CPSC 103/110/APSC 160/EOSC 211/MATH 210/PHYS 210/COMM 337 之一
CPSC 210
Software Construction
软件构造
4 学分核心编程
Design, development, and analysis of robust software components. Topics include software design, computational models, data structures, debugging, and testing.
健壮软件组件的设计、开发与分析：软件设计、计算模型、数据结构、调试和测试。CS专业核心课。
前置: CPSC 107 或 CPSC 110
CPSC 213
Introduction to Computer Systems
计算机系统导论
4 学分系统
Software architecture, operating systems, I/O architectures; critical sections, deadlock avoidance, performance; disks and networks.
软件架构、操作系统、I/O架构；临界区、死锁避免、性能；磁盘和网络。
前置: CPSC 121 + CPSC 210
CPSC 221
Basic Algorithms and Data Structures
基础算法与数据结构
4 学分理论/算法
Design and analysis of basic algorithms and data structures; algorithm analysis, searching/sorting, data structures, graphs, concurrency.
基础算法与数据结构的设计与分析：算法分析、搜索/排序、数据结构、图、并发。
前置: CPSC 210/CPEN 221 + (CPSC 121 或 MATH 220 或 MATH 226≥68%)
CPSC 259
Data Structures and Algorithms for Electrical Engineers
电气工程师数据结构与算法
4 学分核心编程
Advanced procedural programming. Fundamental algorithms for sorting and searching. Data structures (lists, trees, hash tables), scripting languages, file I/O.
高级过程式编程。排序与搜索基础算法。数据结构（列表、树、哈希表）、脚本语言、文件I/O。
前置: APSC 160
CPSC 298
Co-operative Work Placement I
Co-op实习I
3 学分其他
Approved and supervised technical work experience in the computing industry for a minimum of 3.5 months. Technical report required.
在计算行业经批准的受监督技术工作，至少3.5个月。需提交技术报告。
前置: 限CS Co-op项目学生
CPSC 299
Co-operative Work Placement II
Co-op实习II
3 学分其他
Approved and supervised technical work experience. Normally summer after second year.
经批准的技术工作，通常在二年级后暑期。
前置: CPSC 298

CPSC 302
Numerical Computation for Algebraic Problems
代数问题数值计算
3 学分数值计算
Numerical techniques for basic mathematical processes involving no discretization. Linear systems, round-off errors, norms, eigenvalue problems, nonlinear equations.
不涉及离散化的基本数学过程数值技术：线性系统、舍入误差、范数、特征值问题、非线性方程。
前置: CPSC 103/110/CPEN 221/EOSC 211/PHYS 210 + MATH 101级 + MATH 152/221/223
CPSC 303
Numerical Approximation and Discretization
数值逼近与离散化
3 学分数值计算
Numerical techniques for basic mathematical processes involving discretization. Interpolation, approximation, splines, least squares, numerical differentiation/integration, ODEs.
涉及离散化的基本数学过程数值技术：插值、逼近、样条、最小二乘、数值微分/积分、常微分方程。
前置: 同CPSC 302
CPSC 304
Introduction to Relational Databases
关系数据库导论
3 学分数据库
Overview of database systems, ER models, logical database design and normalization, formal query languages, SQL, data warehouses.
数据库系统概览、ER模型、逻辑数据库设计与规范化、形式查询语言、SQL、数据仓库。
前置: CPSC 221 或 DSCI 221
CPSC 310
Introduction to Software Engineering
软件工程导论
4 学分软件工程
Specification, design, validation, evolution and construction of modern software systems, within socially/professionally relevant domains like ethics, IP, and information security.
现代软件系统的规格说明、设计、验证、演化与构建，涵盖伦理、知识产权、信息安全等领域。
前置: CPSC 213 + CPSC 221
CPSC 311
Definition of Programming Languages
程序设计语言定义
3 学分编程语言
Comparative study of advanced programming language features. Statement types, data types, variable binding, parameter passing. Syntactic and semantic description methods.
高级编程语言特性的比较研究：语句类型、数据类型、变量绑定、参数传递。语法与语义描述方法。
前置: CPSC 210
CPSC 312
Functional and Logic Programming
函数式与逻辑编程
3 学分编程语言
Principles of symbolic computing, using languages based upon first-order logic and the lambda calculus. AI and knowledge representation applications.
符号计算原理，使用基于一阶逻辑和λ演算的语言。AI与知识表示应用。
前置: CPSC 210 或 CPEN 221
CPSC 313
Computer Hardware and Operating Systems
计算机硬件与操作系统
3 学分系统
Instruction sets, pipelining, code optimization, caching, virtual memory management, dynamically linked libraries, exception processing, execution time.
指令集、流水线、代码优化、缓存、虚拟内存管理、动态链接库、异常处理、执行时间。

CPSC 314

人类视觉与色彩；几何变换；2D与3D图形算法；硬件与架构；着色与光照；动画。
