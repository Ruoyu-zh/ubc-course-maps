📘 100/200级

6 courses

ELEC 211
Computing Systems I
计算系统I
5 学分
Boolean algebra; combinational/sequential circuits; microcomputer organization; memory addressing; instruction sets; assembly language; I/O; digital system design.
布尔代数；组合/时序电路；微机组织；内存寻址；指令集；汇编语言；I/O；数字系统设计。
前置: APSC 160
ELEC 212
Computing Systems II
计算系统II
4 学分
Hardware-software interface abstractions: procedure invocation, dynamic dispatch, virtual memory, heap, garbage collection, caches, interrupts, processes, threads, locks, files, devices, networks.
硬件-软件接口抽象：过程调用、动态分派、虚拟内存、堆、垃圾回收、缓存、中断、进程、线程、锁、文件、设备、网络。
前置: CPEN 211 + CPEN 221/223/CPSC 259
ELEC 221
Software Construction I
软件构造I
4-5 学分
Design, implementation, reasoning about software: abstraction, specification, testing, verification, abstract data types, OOP, type hierarchies, concurrent design.
软件设计、实现与推理：抽象、规格说明、测试、验证、抽象数据类型、OOP、类型层次、并发设计。
前置: APSC 160
ELEC 223
Software Design for Engineers I
工程师软件设计I
4 学分
Robust software design: static typing, procedural specifications, unit testing, black/white-box tests, user-defined data types, data abstractions, UML.
健壮软件设计：静态类型、过程式规格、单元测试、黑/白盒测试、用户定义类型、数据抽象、UML。
前置: APSC 160
ELEC 281
Technical Communication
技术沟通
3 学分
Written and oral communication in engineering.
工程书面与口头沟通。
前置: APSC 176/ENGL 110/111/112/120/121/WRDS 150
ELEC 291
CE Design Studio I
计算机工程设计工作室I
6 学分
Design projects involving hardware and software.
涉及硬件和软件的设计项目。
前置: CPEN 211
并修/注意: 并修: ELEC 201

📙 300级

8 courses

ELEC 311
Digital Systems Design
数字系统设计
4 学分
Advanced combinational/sequential electronic system design using HDLs and CAD; FPGAs; complex state machines; microcontrollers; arithmetic circuits; interface units.
使用HDL和CAD的高级组合/时序电子系统设计；FPGA；复杂状态机；微控制器；算术电路；接口单元。
前置: CPEN 211 或 CPEN 312
ELEC 312
Digital Systems and Microcomputers
数字系统与微机
3 学分
Data representation; Boolean algebra; combinatorial/sequential circuits; memory/PLDs; microcomputer organization; I/O interfacing.
数据表示；布尔代数；组合/时序电路；存储器/PLD；微机组织；I/O接口。
前置: ELEC 201/203/204 或 MECH 221
ELEC 320
Software Construction II
软件构造II
4 学分
Contemporary concepts for interactive software: client-server architectures, session-oriented systems, security, scalability, deployment models.
交互式软件当代概念：客户端-服务器架构、面向会话系统、安全、可扩展性、部署模型。
前置: CPEN 221 + CPSC 221
ELEC 321
Software Engineering
软件工程
4 学分
Engineering practices for non-trivial software-intensive systems: requirements, architecture, implementation, verification, maintenance, iterative development.
非平凡软件密集系统工程实践：需求、架构、实现、验证、维护、迭代开发。
并修/注意: 并修: CPEN 322
ELEC 331
Operating Systems
操作系统
4 学分
Operating systems design/implementation: process concurrency, synchronization, scheduling, device drivers, memory management, virtual memory, file systems, networking, security.
操作系统设计与实现：进程并发、同步、调度、设备驱动、内存管理、虚拟内存、文件系统、网络、安全。
前置: CPEN 212 + CPEN 221
ELEC 333
Software Design for Engineers II
工程师软件设计II
3-4 学分
OS abstractions; real-time systems; concurrent programming; information structures; OO analysis/design/modelling with UML; software testing. Not open to CE students.
OS抽象；实时系统；并发编程；信息结构；面向对象分析/设计/UML建模；软件测试。非CE学生。
前置: CPSC 259 或 CPEN 223
ELEC 355
ML with Engineering Applications
机器学习工程应用
4 学分
Foundations of data science and ML with applications to engineering problems.
数据科学与机器学习基础及其工程应用。
前置: MATH 152/221 + MATH 302/318/STAT + CPEN 221/223/CPSC 259
ELEC 391
CE Design Studio II
计算机工程设计工作室II
6 学分
Design/implementation of hardware platform + software at OS/application layers; project management; presentations.
硬件平台+软件(OS/应用层)的设计与实现；项目管理；演示。
前置: CPEN 212 + CPSC 221 + ELEC 201

📕 400级

15 courses

ELEC 411
Computer Architecture
计算机架构
4 学分
Modern processors, GPUs, memory hierarchies; instruction set design; pipelining; superscalar; out-of-order; branch prediction; caches; virtual addressing; multicores; VLIW; on-chip networks.
现代处理器、GPU、存储层次；指令集设计；流水线；超标量；乱序执行；分支预测；缓存；虚拟寻址；多核；VLIW；片上网络。
前置: CPEN 211 + CPEN 212
ELEC 412
Microcomputer Systems Design
微机系统设计
4 学分
Microprocessor/system buses; advanced I/O; interrupts; serial I/O; networking; memory system design; hardware/software interaction; testability; safety critical systems.
微处理器/系统总线；高级I/O；中断；串行I/O；网络；存储系统设计；软硬件交互；可测试性；安全关键系统。
前置: CPEN 311 + CPEN 211/312
ELEC 416
Gate-Model Quantum Computing
门模型量子计算
4 学分
Software-based introduction to gate model quantum computing: Grover's, Shor's, quantum compilation, variational algorithms, Hamiltonian simulation, noise characterization.
门模型量子计算软件导论：Grover算法、Shor算法、量子编译、变分算法、哈密顿模拟、噪声表征。
前置: CPEN 221/223/CPSC 221/259/PHYS 210 + MATH 152/221/223
ELEC 421
Software Project Management
软件项目管理
4 学分
Advanced project planning, cost estimation, scheduling; project management tools; productivity factors/metrics; risk analysis; case studies.
高级项目规划、成本估算、调度；项目管理工具；生产力因素/度量；风险分析；案例研究。
前置: CPEN 321 或 CPSC 310
ELEC 422
Software Testing and Analysis
软件测试与分析
4 学分
Unit, integration, system, performance, regression testing; requirements flowdown; coverage criteria; static methods; tools; correctness, reliability, safety assessment.
单元/集成/系统/性能/回归测试；需求分解；覆盖准则；静态方法；工具；正确性/可靠性/安全性评估。
前置: CPEN 221/321 或 CPSC 210/310
ELEC 423
Software Architecture
软件架构
4 学分
Software architecture bridging requirements to implementation; architectural description/patterns; iterative development; COTS products.
连接需求到实现的软件架构；架构描述/模式；迭代开发；商用现货产品。
前置: CPEN 221/321 或 CPSC 310
ELEC 431
Design of Distributed Software Applications
分布式软件应用设计
4 学分
Communications, processes, naming, synchronization, consistency/replication, fault tolerance, middleware, security for distributed applications.
分布式应用的通信、进程、命名、同步、一致性/复制、容错、中间件、安全。
前置: CPEN 331/CPSC 313 + CPEN 221/CPSC 210
ELEC 432
Real-time System Design
实时系统设计
4 学分
Multi-tasking; interrupt-driven systems; task scheduling; schedulability analysis; IPC/synchronization; resource management; performance; hardware/software integration; reliability.
多任务；中断驱动系统；任务调度；可调度性分析；IPC/同步；资源管理；性能；软硬件集成；可靠性。
前置: CPEN 331/333 或 CPSC 313
ELEC 441
Human Computer Interfaces in Engineering Design
工程设计中的HCI
4 学分
Task analysis; user modelling; usability engineering; representations; metaphors; prototyping tools; interactive multimedia; scientific/engineering visualization.
任务分析；用户建模；可用性工程；表示；隐喻；原型工具；交互多媒体；科学/工程可视化。
前置: CPSC 221 + CPEN 221/CPSC 210/CPEN 333
ELEC 442
Introduction to Cybersecurity
网络安全导论
4 学分
Security risks, threats, vulnerabilities; cryptography; software/web security; access control; usable privacy/security; secure systems engineering; cryptocurrencies.
安全风险/威胁/漏洞；密码学；软件/Web安全；访问控制；可用隐私/安全；安全系统工程；加密货币。
前置: CPEN 331/333 或 CPSC 313
ELEC 455
Deep Learning
深度学习
4 学分
Fundamentals of deep learning: MLPs, CNNs, RNNs, Transformers, GNNs; supervised, unsupervised, reinforcement learning.
深度学习基础：MLP、CNN、RNN、Transformer、GNN；监督/无监督/强化学习。
前置: MATH 152/221 + MATH 302/318/STAT + CPEN 221/223/CPSC 259
并修/注意: 推荐: CPEN 355 或 CPSC 340
ELEC 481
Economic Analysis of Engineering Projects
工程项目经济分析
3 学分
Time-money relationships; economic analysis; interest, inflation, depreciation, taxation, uncertainty; cost estimation; financial analysis.
时间-金钱关系；经济分析；利率、通胀、折旧、税收、不确定性；成本估算；财务分析。
ELEC 491
CE Capstone Design Project
计算机工程顶点设计项目
5-10 学分
Design, analysis, implementation of solutions for real-world CE problems from industry/research labs. Project management.
应对真实计算机工程问题的设计、分析与实现。项目管理。
前置: CPEN 331 + CPSC 320 + CPEN 311/322 + CPEN 291/391
ELEC 497
Research Experience in CE
计算机工程研究经历
1-3 学分
Directed research experience. Pass/Fail.
指导研究经历。通过/不通过。
前置: 三年级以上
ELEC 499
Undergraduate Thesis
本科论文
3-6 学分
Project in EE or CE involving design, experimental and/or computer simulation. Individual thesis.
涉及设计、实验和/或计算机仿真的EE或CE项目。个人论文。
