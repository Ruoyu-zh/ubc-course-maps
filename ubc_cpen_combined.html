📘 100/200级

CPEN 211
Computing Systems I
计算系统I
5 学分
Boolean algebra; combinational/sequential circuits; microcomputer organization; assembly; I/O.
布尔代数；组合/时序电路；微机组织；汇编；I/O。
前置: APSC 160
CPEN 212
Computing Systems II
计算系统II
4 学分
Hardware-software interface: procedure invocation, virtual memory, caches, interrupts, processes, threads.
硬软件接口：过程调用、虚拟内存、缓存、中断、进程、线程。
前置: CPEN 211+CPEN 221/223/CPSC 259
CPEN 221
Software Construction I
软件构造I
4-5 学分
Design/implementation: abstraction, specification, testing, OOP, type hierarchies, concurrency.
设计与实现：抽象、规格说明、测试、OOP、类型层次、并发。
前置: APSC 160
CPEN 223
Software Design for Engineers I
工程师软件设计I
4 学分
Robust software: static typing, unit testing, black/white-box tests, data abstractions, UML.
健壮软件：静态类型、单元测试、黑白盒测试、数据抽象、UML。
前置: APSC 160
CPEN 281
Technical Communication
技术沟通
3 学分
Written and oral communication in engineering.
工程书面与口头沟通。
前置: APSC 176/ENGL/WRDS 150
CPEN 291
CE Design Studio I
CE设计工作室I
6 学分
Design projects involving hardware and software.
涉及硬件和软件的设计项目。
前置: CPEN 211
并修/注意: 并修: ELEC 201

📙 300级

CPEN 311
Digital Systems Design
数字系统设计
4 学分
Advanced digital design with HDLs/FPGAs; complex state machines; microcontrollers; arithmetic circuits.
高级数字设计 HDL/FPGA；复杂状态机；微控制器；算术电路。
前置: CPEN 211/312
CPEN 312
Digital Systems and Microcomputers
数字系统与微机
3 学分
Boolean algebra; combinatorial/sequential circuits; memory/PLDs; microcomputer organization; I/O.
布尔代数；组合/时序电路；存储器/PLD；微机组织；I/O。
前置: ELEC 201/203/204 or MECH 221
CPEN 320
Software Construction II
软件构造II
4 学分
Interactive software: client-server, session-oriented, security, scalability, deployment models.
交互式软件：客户端-服务器、面向会话、安全、可扩展性、部署。
前置: CPEN 221 + CPSC 221
CPEN 321
Software Engineering
软件工程
4 学分
Requirements, architecture, implementation, verification, maintenance, iterative development.
需求、架构、实现、验证、维护、迭代开发。
并修/注意: 并修: CPEN 322
CPEN 331
Operating Systems
操作系统
4 学分
OS design: process concurrency, synchronization, scheduling, memory management, file systems, networking.
OS设计：进程并发、同步、调度、内存管理、文件系统、网络。
前置: CPEN 212 + CPEN 221
CPEN 333
Software Design for Engineers II
工程师软件设计II
3-4 学分
OS abstractions; real-time systems; concurrent programming; OO design with UML; software testing.
OS抽象；实时系统；并发编程；UML面向对象设计；软件测试。
前置: CPSC 259/CPEN 223
CPEN 355
ML with Engineering Applications
ML工程应用
4 学分
Foundations of data science and ML with engineering applications.
数据科学与ML基础及工程应用。
前置: MATH 152/221+STAT+CPEN 221/223/CPSC 259
CPEN 391
CE Design Studio II
CE设计工作室II
6 学分
Hardware platform + software at OS/application layers; project management; presentations.
硬件平台+软件(OS/应用层)；项目管理；演示。
前置: CPEN 212+CPSC 221+ELEC 201

📕 400级

CPEN 411
Computer Architecture
计算机架构
4 学分
Modern processors, GPUs, memory hierarchies; pipelining; superscalar; caches; multicores; VLIW.
现代处理器、GPU、存储层次；流水线；超标量；缓存；多核；VLIW。
前置: CPEN 211+CPEN 212
CPEN 412
Microcomputer Systems Design
微机系统设计
4 学分
Microprocessor/system buses; advanced I/O; interrupts; networking; memory design.
微处理器/系统总线；高级I/O；中断；网络；存储设计。
前置: CPEN 311+CPEN 211/312
CPEN 416
Gate-Model Quantum Computing
门模型量子计算
4 学分
Grover, Shor algorithms; quantum compilation; variational algorithms; Hamiltonian simulation.
Grover/Shor算法；量子编译；变分算法；哈密顿模拟。
前置: CPEN/CPSC programming+MATH 152/221/223
CPEN 421
Software Project Management
软件项目管理
4 学分
Advanced project planning, cost estimation, scheduling; risk analysis; case studies.
高级项目规划、成本估算、调度；风险分析；案例研究。
前置: CPEN 321/CPSC 310
CPEN 422
Software Testing and Analysis
软件测试与分析
4 学分
Unit/integration/system/performance testing; coverage criteria; static methods.
单元/集成/系统/性能测试；覆盖准则；静态方法。
前置: CPEN 221/321/CPSC 210/310
CPEN 423
Software Architecture
软件架构
4 学分
Architecture bridging requirements to implementation; patterns; iterative development; COTS.
连接需求到实现的架构；模式；迭代开发；商用现货。
前置: CPEN 221/321/CPSC 310
CPEN 431
Distributed Software Applications
分布式软件应用
4 学分
Communications, processes, naming, synchronization, replication, fault tolerance, middleware.
通信、进程、命名、同步、复制、容错、中间件。
前置: CPEN 331/CPSC 313+CPEN 221/CPSC 210
CPEN 432
Real-time System Design
实时系统设计
4 学分
Multi-tasking; interrupt-driven; task scheduling; schedulability; IPC; resource management.
多任务；中断驱动；任务调度；可调度性；IPC；资源管理。
前置: CPEN 331/333/CPSC 313
CPEN 441
Human Computer Interfaces in Engineering
工程设计HCI
4 学分
Task analysis; user modelling; usability engineering; prototyping; interactive multimedia.
任务分析；用户建模；可用性工程；原型；交互多媒体。
前置: CPSC 221+CPEN 221/CPSC 210/CPEN 333
CPEN 442
Introduction to Cybersecurity
网络安全导论
4 学分
Security risks/threats/vulnerabilities; cryptography; software/web security; access control.
安全风险/威胁/漏洞；密码学；软件/Web安全；访问控制。
前置: CPEN 331/333/CPSC 313
CPEN 455
Deep Learning
深度学习
4 学分
MLPs, CNNs, RNNs, Transformers, GNNs; supervised, unsupervised, reinforcement learning.
MLP/CNN/RNN/Transformer/GNN；监督/无监督/强化学习。
前置: MATH 152/221+STAT+CPEN 221/223/CPSC 259
CPEN 491
CE Capstone Design Project
CE顶点设计
5-10 学分
Design/analysis/implementation for real-world CE problems. Project management.
真实CE问题的设计/分析/实现。项目管理。
前置: CPEN 331+CPSC 320+CPEN 311/322+CPEN 291/391
CPEN 499
Undergraduate Thesis
本科论文
3-6 学分
Project in EE or CE: design, experimental and/or computer simulation.
EE或CE项目：设计、实验和/或计算机仿真。
